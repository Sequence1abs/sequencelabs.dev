## Implementation Notes

**OG Images** — Create a 1200x630px branded image for each page and place them at the paths listed above. A simple card with your logo and page name on a dark background works well.

**Schema markup** — Paste each JSON-LD block inside a `<script type="application/ld+json">` tag in the `<head>` of the corresponding page.

**SPF / DKIM / DMARC** — Ensure these DNS records are set in Cloudflare so emails from hello@sequencelabs.dev do not land in spam.

**Canonical tags** — Always use the exact URL with no trailing slashes inconsistency. Pick one format and stick to it across all pages.

**Title character limits** — Keep titles between 40 and 60 characters. Google truncates anything beyond 60.

**Description character limits** — Keep descriptions between 120 and 160 characters for best display in search results.
